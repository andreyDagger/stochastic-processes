Тема проекта: Моделирование динамики общественного мнения в социальных сетях с использованием дискретных цепей Маркова

Авторы: Матвеев Андрей M3439

Цель и задачи проекта:
Цель проекта: Исследовать с помощью математического моделирования, как свойства социальной сети и характеристики контента влияют на масштаб и скорость распространения информации.

Задачи:

Разработать стохастическую модель пользователей социальной сети на основе дискретной цепи Маркова.

Реализовать генерацию графов социальных сетей с различной структурой

Провести серию вычислительных экспериментов для анализа влияния параметров модели и структуры сети на динамику процесса.

Визуализировать результаты.

Применяемые модели и методы
Основная модель случайного процесса: Неоднородная дискретная цепь Маркова. Каждый пользователь моделируется как автомат, состояние которого в следующий момент времени зависит только от его текущего состояния и состояний его соседей.

Пространство состояний пользователя: Неосведомленный, Распространитель, Скептик.

Модель социальной сети: Ориентированный или неориентированный граф..

Математический аппарат: Для описания переходов используются вероятности, зависящие от количества соседей в определенных состояниях, что делает процесс неоднородным Марковским процессом на графе.
 
Ожидаемые результаты
Имитационная модель, написанная на языке Python.

Визуализация кривой роста числа информированных пользователей.

Демонстрация порогового эффекта: при низкой "заражающей" способности контента, распространение затухает, а при большой – заражает почти всех..
